🔮 ORB

Emotional Companion · CS291 Spring 2026

What It Does

An AI companion that senses your emotional state through voice tone analysis and responds — comforting you when sad, celebrating when happy, giving space when stressed. It doesn't wait for commands. It notices.

Tech Stack

  • Three.js — 3D rendering + particle system
  • Web Audio API — real-time voice analysis
  • Claude API — AI decision engine
  • IndexedDB — mood journal + session history
  • PWA — installable, works offline
  • Custom ML — pitch/energy/cadence classifier

Features

  • Real-time voice emotion detection
  • Mood journal with session history
  • Exportable mood data (JSON)
  • Works on any device with a mic

Team

  • Harb — hardware + software integration
  • Kaiyuan Duan — emotion detection + tuning
