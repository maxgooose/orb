🔮 ORB

Emotional Companion Ball · CS291 Spring 2026

What It Does

A rolling ball that senses your emotional state through voice tone analysis and physically responds — rolling toward you when sad, spinning when happy, backing away when stressed. It doesn't wait for commands. It notices.

Tech Stack

  • Raspberry Pi Zero 2W — on-device control
  • Claude API — AI decision engine
  • Python — audio processing + motor control
  • MEMS microphone — voice tone detection
  • NeoPixel LEDs — mood-mapped color response
  • Pendulum drive — DIY rolling mechanism
